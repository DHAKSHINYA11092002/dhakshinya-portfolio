# EdTech Learner Analytics

This project analyzes 50,000+ synthetic learner activity logs and builds a dropout prediction model using Logistic Regression.

## Files Included
- learner_activity.csv — synthetic dataset
- eda.ipynb — notebook with EDA, feature engineering, modeling
- logistic_model.joblib — saved ML model (logistic regression)

## Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Visualization
- Logistic Regression Model
- Student Dropout Prediction
